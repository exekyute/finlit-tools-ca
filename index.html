Why these exist

I built these to refresh hands-on coding fundamentals while practicing the harder skill: explaining dense Canadian tax-and-benefit rules in plain language, with the rules themselves as the constraint that forces clarity. Each tool is one HTML file — no build step, no server, no install. Double-click to run, or use the hosted version.

Disclaimer & sources

These tools are illustrative only. They use 2026-indexed estimates of federal and provincial rules and simplify many real-world variables. Each tool documents what it does and does not model in its own footer.
